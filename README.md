# Employee Management System

The **Employee Management System** is a desktop-based application developed using Python and Tkinter, designed to efficiently manage employee records within an organization.  
This project allows administrators to add, update, delete, search, and view employee details with a user-friendly graphical interface and a MySQL backend.

The system is developed as a **Team Project under Team.XLR8** and focuses on simplicity, data accuracy, and real-world usability.

## Tech Stack
- **Programming Language:** Python  
- **GUI Framework:** Tkinter  
- **Database:** MySQL  
- **Image Handling:** Pillow (PIL)  
- **Data Handling:** CSV  

---

## Features
- Secure login system for admin access  
- Add new employee records  
- Update existing employee information  
- Delete employee records  
- Search employees by ID or phone number  
- Display employee data in tabular format  
- CSV data import into MySQL database  
- Interactive and user-friendly Tkinter GUI  
- Image-based branding using Tkinter assets  

---

## Project Structure
- `main.py` – Main application file  
- `assets/` – Contains images used in the GUI  
- `employee_data.csv` – Sample employee dataset  
- `README.md` – Project documentation  

---

## How to Run the Project
1. Install required dependencies:
    " pip install pillow mysql-connector-python "
2. Make sure MySQL is installed and running  
3. Update MySQL credentials in `main.py`
4. Run the application:
    " python main.py "

---

## Note
- Keep the `assets` folder in the same directory as `main.py`
- Update database credentials before running the project
